# ASXcodeSourcePlugin

## 介绍
#### 用于自动化生成Masnory代码和实现懒加载等。代码持续维护中，作为工作中的插件小工具使用。

## 下载安装
- 如果之前编译过 ASXcodeSourcePlugin，清除 DrivedData
- 在这里安装：[源码地址](https://github.com/iAlexSun/ASXcodeSourceExtensioin)
- 打开 Xcode 并选择文本
- 在 Editor 菜单中找到插件
- 你可以给每个插件都设置一个独立的快捷键
